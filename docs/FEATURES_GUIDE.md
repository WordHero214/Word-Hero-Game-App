- Collect sparkies

## â“ FAQ

**Q: What happens if I miss a day?**
A: Your streak resets to 1, but you can start building again immediately.

**Q: Do Quick Play games count toward my streak?**
A: Yes! Any game mode counts for your daily streak.

**Q: Can I practice wrong words multiple times?**
A: Absolutely! Practice as much as you want to improve.

**Q: How long does progress history keep data?**
A: The last 30 days are stored automatically.

**Q: Do I lose sparkies if I get words wrong?**
A: No! You only earn sparkies, never lose them.
