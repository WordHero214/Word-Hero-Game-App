## Troubleshooting

### Problem: AI stops generating mid-way
**Solution**: Type "Please continue" and it will resume

### Problem: Wrong format
**Solution**: Say "Please format as CSV with these exact columns: Word,Difficulty,Grade,Hint (English),Scenario (English),Hint (Filipino),Scenario (Filipino)"

### Problem: Words too easy/hard
**Solution**: Say "Please make Grade [X] words more appropriate for [age] year olds"

### Problem: Filipino translations unclear
**Solution**: Say "Please revise Filipino translations to use simpler, everyday Tagalog"

### Problem: Can't save as CSV
**Solution**: 
1. Save as .txt first
2. Close file
3. Rename from .txt to .csv
4. Windows will warn you - click "Yes"
