## üêõ Troubleshooting

**Words not appearing for students?**
- Check grade level and section assignments
- Verify student's grade/section matches word assignments
- Use Word Debugger (üêõ button) to see loaded words

**Certificate not awarded?**
- Verify 10+ words exist for that difficulty
- Check if student got 100% score
- Look at browser console for certificate logs

**Timer not working?**
- Refresh the page
- Check browser console for errors
- Verify you're using latest code
